字段内容
根因特征信号覆盖率低（零值率78.6%），且各行业零值率差异小，属于合法零值（管理层确实未使用转折连接词），导致信号稀疏、预测能力被大量零值稀释，IC虽显著但实际应用受限。
修复建议将特征定义从“转折连接词频率”改为“转折连接词与正面/负面情感词的交互频率”，例如统计“but + 负面词”或“however + 正面词”的共现频率，以捕捉管理层刻意平衡信息的意图，提高信号区分度。同时，放宽condition_scope中的section_type，仅保留“qa”部分（问答环节更易出现转折性解释），减少prepared部分中低信息密度的零值。
避免不要直接降低零值率阈值或对零值样本进行插值填充，这会引入噪声并破坏合法零值的真实语义。

❌ v1 (原始) — FAIL

问题: 基于具体词频作离散化分档 → 大量样本集中在1-3/千词区间(得0分) → zr=78.6%

✅ v2 (修复) — PASS

Assess the frequency and intensity of contrastive language (e.g., expressions of contrast, concession, or hedging) used by management in the transcript. Score based on the overall semantic impression rather than exact word counts. Use the full range -2 to +2: -2 for very frequent and strong contrastive tone, -1 for frequent, 0 for genuinely balanced with no clear leaning, +1 for infrequent, +2 for almost absent. Score must use the full range. Reserve 0 only for genuinely balanced text.
修复: 基于整体语义印象连续打分 → 非零样本大增 → zr=44.5%
